# Movie Data Analysis (IMDb Dataset)

This project performs exploratory data analysis (EDA) on an IMDb movie dataset using Python.

## Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn (Basic)

## Key Work
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Visualization of ratings, votes, revenue, and genres
- Genre-based analysis

## Files
- Movie analysis.ipynb → Main analysis notebook
- imdb_movie_dataset.csv → Dataset
- Movie analysis.pdf → Project report
